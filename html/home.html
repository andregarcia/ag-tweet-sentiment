Performing Sentiment Analysis on Twitter data

What is Tweet Sentiment?

Tweet Sentiment is a web application that performs Sentiment Analysis on Twitter data making use of the IBM's AlchemyAPI. Currently there is only one search mode: by term. The user (you!) provides the search term. The application will search for popular twitter posts (using the Twitter API) containing the given term and will then perform the sentiment analysis on the results.
