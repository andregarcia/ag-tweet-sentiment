Tweet Sentiment

Performing Sentiment Analysis on Twitter data

Author

The author of this project is Andre Garcia.

Source code

The source code of this project can be found on github: https://github.com/andregarcia/ag-tweet-sentiment

Used Technologies and Resources

Here is the list of technologies and resources that were used in this project:

  • Python's Flask web framework: used to develop the backend of the web application.
  • IBM's AlchemyAPI: used to perform sentiment analysis on text.
  • IBM's Bluemix: cloud solution used to deploy and host this web application.
  • Twitter API: used to retrieve twitter posts.
  • Python's twitter lib: used to access the twitter api through python code.
  • Yahoo Pure CSS template: used as layout for the web pages. The used template can be found here.
